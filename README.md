

# Portfolio Website
======================

A personal portfolio website built with Next.js, React, and Tailwind CSS.

## Overview
-----------

This project is a personal portfolio website that showcases my skills, experience, and projects. It is built using Next.js, a popular React-based framework for building server-side rendered (SSR) and statically generated websites.

## Features
------------

* **Home**: A hero section that introduces myself and my skills.
* **About**: A section that provides more information about my background, education, and experience.
* **Portfolio**: A section that showcases my projects, including images, descriptions, and links to live demos and GitHub repositories.
* **Skills**: A section that lists my technical skills and expertise.
* **Contact**: A section that provides contact information, including email and phone number.
* **Footer**: A section that includes links to my social media profiles and a copyright notice.

## Technologies Used
----------------------

* **Next.js**: A React-based framework for building SSR and statically generated websites.
* **React**: A JavaScript library for building user interfaces.
* **Tailwind CSS**: A utility-first CSS framework for building custom user interfaces.
* **Framer Motion**: A library for building animations and gestures.
* **PostCSS**: A tool for transforming CSS with JavaScript.

## Getting Started
-------------------

To get started with this project, follow these steps:

1. Clone the repository: `git clone https://github.com/ikhwanand/portfolio-next-js.git`
2. Install dependencies: `npm install` or `yarn install`
3. Start the development server: `npm run dev` or `yarn dev`
4. Open the website in your browser: `http://localhost:3000`

## Building and Deployment
---------------------------

To build and deploy the website, follow these steps:

1. Build the website: `npm run build` or `yarn build`
2. Deploy the website: `npm run start` or `yarn start`
